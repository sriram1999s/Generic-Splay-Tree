# Splay Tree

## Functionalities

- Insertion
- Removal
- Joining
- Splitting
- Finding
- Iteration

## iterator

- bidirectional
- ++, --, ==, !=, =, *
- All std algorithms for bidirectional iterator

# Usage

## Inclusion

```cpp
#include "splay_tree.hpp"
```

## Declaration

```cpp
SplayTree<int> st1;
SplayTree<float> st2;
```
